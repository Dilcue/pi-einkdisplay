# Security Policy

## Scope

This is a personal hobby project designed to run on a trusted home LAN. The web UI has no authentication by design and should never be exposed to an untrusted network or the public internet.

## Reporting a Vulnerability

If you find a security issue, please report it privately using [GitHub's private vulnerability reporting](https://github.com/Dilcue/pi-einkdisplay/security/advisories/new) rather than opening a public issue.

There is no SLA or bug bounty — this is a hobby project — but reports are appreciated and will be addressed on a best-effort basis.
